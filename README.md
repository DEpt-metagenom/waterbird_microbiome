On this site, we uploaded the supplementary material for the manuscript titled: Microbiota shows major difference in case of two shorebird species with different feeding strategy. Authored by Ákos Őrsi, Levente Laczkó, Renáta Bőkényné Tóth, Csongor Freytag, Pál Tóth, Gábor Simay, Nándor Szabó, Gábor Kardos, Ádám Lovas-Kiss.

waterbird_1, waterbird_2, waterbird_3: these folders contain the abundance tables and the positive controls for each PCR run.

Data_abundances.xlsx: This Excel file contains the final merged abundances of each PCR run.

get_copy_numbers.R: This R script normalises 16 copy numbers based on 16s known copy numbers for each run.

get_copy_numbers_merged.R: This R script normalises 16 copy numbers based on 16s known copy numbers for the together run.

merged_wider.csv: This CSV file was used for the statistical community analysis. This contains the merged corrected abundances for each sample.

plot_err.R: This R script helps to check for errors and compare with the positive controls.

samples_metadata.xlsx: This Excel file contains the metadata for the collected and used samples

final_analysis.R: This R script contains the community analysis of the microbiome
